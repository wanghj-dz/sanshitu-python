═══════════════════════════════════════════════════════

【方式一】直接运行Python程序(需要Python环境)
───────────────────────────────────────────────────────
1. 确保已安装Python 3.11或更高版本
2. 打开命令提示符(cmd)或终端
3. 进入程序目录
4. 运行命令:
   pip install numpy matplotlib
   python projection_experiment.py

【方式二】打包成exe文件(Windows系统)
───────────────────────────────────────────────────────
1. 确保已安装Python 3.11或更高版本
2. 双击运行"打包程序.bat"
3. 等待2-3分钟,打包完成
4. 进入dist文件夹,双击"投影实验程序.exe"运行

【方式三】手动打包(高级用户)
───────────────────────────────────────────────────────
1. 打开命令提示符
2. 运行命令:
   pip install pyinstaller numpy matplotlib
   pyinstaller --onefile --windowed --name "投影实验程序" projection_experiment.py
3. 打包完成后,exe文件位于dist文件夹
